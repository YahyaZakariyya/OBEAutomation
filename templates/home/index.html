About OBE

Education Designed Around Outcomes

Outcome-Based Education shifts the focus from what is taught to what students achieve. Every course, assessment, and curriculum decision is anchored to measurable learning outcomes — ensuring graduates meet the competencies that matter.

Outcome-Driven Design

Programs define Program Learning Outcomes (PLOs), and every course CLO maps directly to one PLO — creating a traceable chain from daily classwork to graduate attributes.

Weighted Attainment Tracking

CLOs and PLOs carry explicit weightages (summing to 100%). Assessment questions are tagged to CLOs, and student marks translate directly into attainment percentages.

Continuous Quality Improvement

Real-time dashboards surface attainment gaps for both faculty and administrators — powering data-driven decisions to improve curricula each semester.

Key Concepts

The Building Blocks of OBE

Programs

Academic programs (e.g. BSCS, BSIT, MBA) are the top-level entity. Each program has a unique abbreviation, type (UG/PG), and an assigned incharge faculty.

e.g. BSCS · BSIT · BBA

Program Learning Outcomes (PLOs)

Each program defines 1–15 PLOs — high-level graduate attributes. Every PLO carries a weightage, and the total across a program must equal 100%.

Σ PLO weightages = 100%

Courses

Courses belong to one or more programs. Each course has a code, name, and credit hours (max 4). Shared courses like GEN101 can span multiple programs simultaneously.

max 4 credit hours

Course Learning Outcomes (CLOs)

Each course defines up to 15 CLOs. CLO weightages per course must not exceed 100% cumulatively. The system enforces this at save time.

Σ CLO weightages ≤ 100%

PLO-CLO Mapping

Every CLO maps to exactly one PLO — globally per course. This strict rule ensures attainment can be cleanly rolled up from CLO scores to PLO attainment percentages.

1 CLO → 1 PLO (enforced)

Assessments & Scores

Assessments (Quiz, Midterm, Final…) contain questions, each tagged to a CLO. Student marks on questions feed the attainment calculation pipeline.

Questions → CLOs → PLOs
Weightage System

How Percentages & Weightages Work

The OBE system uses a layered weightage model — from individual assessment marks all the way to program-level attainment. Every number is transparent, traceable, and constrained.

PLO Weightages

Each PLO in a program carries a percentage that reflects its importance in the curriculum. All PLO weightages for a program sum to exactly 100%.

Sum = 100%

CLO Weightages

CLO weightages per course reflect the relative importance of each outcome. The cumulative total across all CLOs in a course must not exceed 100%.

Sum ≤ 100%

Mapping Weightages

When a CLO maps to a PLO, the mapping itself carries a weightage. The sum of mapping weightages per (course, PLO) pair must not exceed 100%.

Per-PLO ≤ 100%

OBE Attainment

Student marks on assessment questions → CLO scores → weighted CLO attainment → PLO attainment. The system computes this pipeline automatically per section.

Auto-computed
